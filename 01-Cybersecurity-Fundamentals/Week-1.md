# 🛡️ Foundation of Cybersecurity

## 📌 Introduction

Cybersecurity is the practice of protecting **computers, networks, applications, systems, devices, and data** from unauthorized access, attacks, damage, or theft.

The main goal of cybersecurity is to maintain:

- **Confidentiality**
- **Integrity**
- **Availability**

These three principles form the foundation of cybersecurity and are known as the **CIA Triad**.

---

# 🎯 Why Cybersecurity is Important

Modern organizations depend on technology for storing and processing information. Cybersecurity protects:

- **Personal Data**
- **Business Information**
- **Financial Records**
- **Banking Systems**
- **Applications**
- **Cloud Services**
- **Networks**
- **Internet Infrastructure**
- **Customer Identity**

A security failure can cause:

- Data breaches
- Financial losses
- Reputation damage
- Service downtime
- Privacy violations

---

# 🔺 CIA Triad

The CIA Triad is the foundation model of information security.

## 🔒 Confidentiality

Ensures information is accessed only by authorized users.

### Protection Methods:

- Passwords
- Multi-Factor Authentication (MFA)
- Encryption
- Access Control

Example:

A customer's banking details should only be visible to authorized people.

---

## ✅ Integrity

Ensures data remains accurate, complete, and trustworthy.

### Protection Methods:

- Hashing
- Digital Signatures
- File Integrity Monitoring
- Version Control

Example:

A transaction amount should not be changed by an unauthorized person.

---

## ⚡ Availability

Ensures systems and information are available whenever required.

### Protection Methods:

- Backup Systems
- Disaster Recovery Plans
- Load Balancing
- Anti-DDoS Protection

Example:

A banking website should remain available for customers.

---

# 🏢 Security Operations Center (SOC)

A Security Operations Center is a team or facility responsible for monitoring and protecting an organization's security environment.

SOC works:

**24/7 Monitoring → Detection → Investigation → Response**

## SOC Responsibilities:

- Monitor security logs
- Detect suspicious activities
- Investigate alerts
- Analyze threats
- Respond to incidents
- Perform threat hunting
- Improve security controls

# 🔐 Cybersecurity Terminology

Understanding basic cybersecurity terms is important for learning security concepts, attacks, and defense mechanisms.

---

# 🦠 Virus

A **virus** is a type of malicious software (malware) that attaches itself to files or programs and spreads when the infected file is executed.

Purpose:

- Steal sensitive information
- Damage files
- Modify system data

Protection:

- Antivirus software
- Regular updates
- Avoid suspicious files

---

# ⚠️ Threat

A **threat** is any possible danger that can exploit a vulnerability and cause harm to a system, network, or data.

Examples:

- Malware attacks
- Phishing attacks
- Data breaches
- Unauthorized access

---

# 📉 Risk

**Risk** is the possibility of damage or loss when a threat exploits a vulnerability.

Risk depends on:

- Threat
- Vulnerability
- Impact

Example:

A weak password (vulnerability) can be exploited by an attacker (threat), causing account theft (impact).

---

# 🔓 Vulnerability

A **vulnerability** is a weakness or security flaw in hardware, software, network, or processes.

Examples:

- Outdated software
- Weak passwords
- Misconfigured systems
- Security bugs

---

# 🔍 Vulnerability Assessment

Vulnerability Assessment is the process of identifying, analyzing, and reporting security weaknesses in a system.

Steps:

1. Discover vulnerabilities
2. Analyze severity
3. Generate reports
4. Recommend fixes

Common Severity Levels:

- Critical
- High
- Medium
- Low

Tools:

- Nessus
- OpenVAS
- Qualys

---

# 🛡️ Penetration Testing

Penetration Testing is an authorized security test performed to identify and exploit vulnerabilities before attackers can use them.

Goals:

- Find security weaknesses
- Test defenses
- Improve security

Stages:

1. Reconnaissance
2. Scanning
3. Enumeration
4. Exploitation
5. Reporting

---

# 🔥 Firewall

A firewall is a security system that monitors and controls incoming and outgoing network traffic.

Functions:

- Allow trusted connections
- Block unauthorized access
- Filter network traffic

Types:

- Network Firewall
- Host Firewall
- Application Firewall

Example:

A firewall acts like a security guard between a private network and the internet.

---

# 🚨 IDS (Intrusion Detection System)

IDS monitors network traffic and detects suspicious activities.

Functions:

- Analyze network behavior
- Detect threats
- Generate alerts

IDS mainly focuses on **detection**.

---

# 🛑 IPS (Intrusion Prevention System)

IPS detects and prevents malicious activities automatically.

Functions:

- Monitor traffic
- Block attacks
- Prevent unauthorized access

IPS focuses on **detection + prevention**.
